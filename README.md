# vk-music-hotkeys

Инструкция:

1.Необходимо включить режим разработчика в настройках браузера.
2.Найдите кнопку "Загрузить распакованное расширение". Выберите путь к расширению (оно должно быть распаковано из архива).
3.Во вкладке "Сочетание клавиш" (может называться иначе) выберите горячие клавиши по вашему усмотрению.

Важно:

В первый раз паузу нужно нажимать вручную, далее до перезагрузки страницы этого делать не требуется. Также сайт иногда "теряет" вашу очередь прослушивания после перезагрузки - учитывайте и это.
Если нужно, чтобы громкость за одно нажатие изменялась больше или меньше, редактируйте файл background.js - вы сразу заметите нужные вам значения.
